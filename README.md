# Raj Trading Frontend

## API configuration

The frontend reads the backend base URL from `VITE_API_BASE_URL`.

- Local development: create `.env.local` or use `.env.example`
- Production build: `.env.production` points to `https://raj-tradin-be.onrender.com`

Example:

```env
VITE_API_BASE_URL=http://localhost:3000
```
